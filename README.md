# AI Photo Pose Coach 📸

AI-powered iOS app that analyzes and suggests photo poses in real-time using computer vision.

## Trending Signals

- 20,267+ impressions on X (Twitter) in 24 hours
- 205 bookmarks, 120 likes — top trending app idea
- Aligned with Siri 3rd-party AI app widgets at Apple WWDC
- Strong TikTok/short-form video viral potential

## Features

- [ ] Real-time AI pose detection via camera
- [ ] Suggestions for better poses (face angle, body position, lighting)
- [ ] Before/after comparison mode
- [ ] Pose library by context (portrait, group, outdoor, event)
- [ ] Shareable results for social media

## Tech Stack

- **iOS**: SwiftUI + AVFoundation
- **AI**: Apple Vision Pro (VNDetectHumanBodyPoseRequest) / MediaPipe
- **Backend**: Supabase (pose library)
- **Monetization**: Freemium + Pro subscription ($4.99/mo)

## Getting Started

```bash
git clone https://github.com/mozitoooo/ai-photo-pose-coach
cd ai-photo-pose-coach
open AIPoseCoach.xcodeproj
```
